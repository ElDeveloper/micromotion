Leap Motion Controller

The Leap Motion controller is an advanced sensor that provides an alternative to the usage of a mouse and keyboard to interact with computer programs. Emperor takes advantage of this modern interface and integrates a set of gestures that will allow the basic interaction of a user with a dataset presented on screen.

While not all of Emperor's features are supported using this controller, the following features are available via Leap Motion gestures:

  • Rotating
  • Zooming
  • Switching between color schemes
  • Re-entering the camera
  • Adding a cursor to the plot
  • Switching between color categories

Demo

The best pointing device

Our fingers are the best and most accurate pointing devices that mankind has ever seen, so if you point at the screen with a single finger, a cursor will be displayed on top of the plot, any of your fingers can function as a pointer.

Showing your data's true colors

Switching between different coloring categories in your mapping file and coloring schemes (continuous or discrete) are now features at your finger tips. Using a swipe gesture (in any direction i. e. top-down, left-right, etc.) you can change the current category used to color the samples and if you use the tap gesture, you can toggle between continuous and discrete coloring schemes.

Coloring samples given a metadata category is usually the easiest way to verify whether or not there are clusters in the data you are visualizing.

Recenter the camera

If after rotating, zooming in and zooming out you just want to go back to the initial position at which the camera was positioned, just tap the screen and Emperor will reset the position of the camera.
